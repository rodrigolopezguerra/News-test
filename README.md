## ¿Qué es este proyecto?

Es una demostración técnica de cómo dos servicios legítimos y bien dirigidos — **Internet Archive (Wayback Machine)** y un agregador de noticias — pueden complementarse mediante integración de APIs para lograr un objetivo concreto: preservar el acceso a contenido informativo.
